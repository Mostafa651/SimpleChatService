API Reference

Complete API documentation for SimpleChatService functions and events.

ChatConfig

Configuration settings for the chat system.

ChatConfig.CHAT_CONFIG = {
    MaxMessages = 50,
    MessageFadeTime = 30,
    AnimationSpeed = 0.3,
    Colors = {
        Background = Color3.fromRGB(25, 25, 25),
        Text = Color3.fromRGB(255, 255, 255),
        Accent = Color3.fromRGB(0, 162, 255)
    }
}

ChatUI Functions

ChatUI.createChatUI(screenGui)

Creates the main chat interface.

Parameters: screenGui - The ScreenGui to parent the chat to

ChatUI.toggleChat(show, animate)

Shows or hides the chat interface.

Parameters: show (boolean), animate (boolean)

Chat Commands

  • /clear - Clear all chat messages
  • /help - Show available commands
  • /time - Show current time
