Simple settings you can customize

ChatConfig.luau Settings

-- Basic chat configuration
local ChatConfig = {
    MaxMessageLength = 200,
    MessageCooldown = 1, -- seconds
    MaxMessagesPerMinute = 10,
    
    -- UI Settings
    ChatBackgroundColor = Color3.fromRGB(25, 25, 25),
    ChatTextColor = Color3.fromRGB(255, 255, 255),
    
    -- Admin Settings
    AdminPrefix = "/",
    AdminList = {}, -- Add admin user IDs here
}

return ChatConfig

üöÄ Basic Usage

How to use SimpleChatService in your scripts

Sending a Message (Client)

-- Get the chat events
local ChatEvents = require(ReplicatedStorage.ChatEvents)

-- Send a message
ChatEvents.SendMessage:Fire("Hello everyone!")

Listening for Messages (Client)

-- Listen for incoming messages
ChatEvents.MessageReceived:Connect(function(playerName, message, timestamp)
    print(playerName .. ": " .. message)
end)

üõ°Ô∏è Admin Commands

Basic moderation tools available to administrators

/kick [player]

Kicks a player from the game

/mute [player]

Prevents a player from sending chat messages

/unmute [player]

Allows a muted player to send messages again

/announce [message]

Sends a server-wide announcement message

Setting Up Admins

Add user IDs to the AdminList in ChatConfig.luau:

AdminList = {123456789, 987654321} -- Replace with actual user IDs
