Code Examples

Learn how to use SimpleChatService with these practical examples.

Basic Setup

How to set up the chat system in your game:

-- In ServerScriptService
local SimpleChatSystem = ReplicatedStorage:WaitForChild("SimpleChatSystem")
local ChatSystemServer = require(SimpleChatSystem.System.ChatSystemServer)

-- Initialize the chat system
ChatSystemServer.initialize()

Custom Commands

Adding your own chat commands:

-- In ChatCommands.luau
local function customCommand(player, args)
    -- Your command logic here
    return "Command executed!"
end

-- Register the command
ChatCommands.registerCommand("/mycommand", customCommand)

Message Filtering

How messages are filtered:

-- Automatic filtering using Roblox TextService
local filteredMessage = MessageHandler.filterText(message, player.UserId)
ChatEvents.sendMessage(player.Name, filteredMessage, "Normal")

Custom Colors

Customizing the chat appearance:

-- In ChatConfig.luau
ChatConfig.CHAT_CONFIG.Colors = {
    Background = Color3.fromRGB(25, 25, 25),
    Text = Color3.fromRGB(255, 255, 255),
    Accent = Color3.fromRGB(0, 162, 255)
}
