Basic Configuration

Customize the chat system for your game

Edit ChatConfig.luau

Open the ChatConfig module and customize these basic settings:

-- Basic chat configuration
local ChatConfig = {
    -- Message Settings
    MaxMessageLength = 200,
    MessageCooldown = 1, -- seconds between messages
    MaxMessagesPerMinute = 10,
    
    -- UI Colors
    ChatBackgroundColor = Color3.fromRGB(25, 25, 25),
    ChatTextColor = Color3.fromRGB(255, 255, 255),
    
    -- Admin Settings
    AdminPrefix = "/",
    AdminList = {}, -- Add your user ID here: {123456789}
}

return ChatConfig

üîß Troubleshooting

Common issues and simple solutions

Chat not appearing?

  • Check that ChatSystemClient is in StarterPlayer > StarterPlayerScripts
  • Make sure ChatSystemServer is in ServerScriptService
  • Look for errors in the Developer Console (F9)

Messages not sending?

  • Check if you're hitting the message cooldown limit
  • Verify the ChatEvents module is in ReplicatedStorage
  • Make sure both server and client scripts are running

Admin commands not working?

  • Add your user ID to the AdminList in ChatConfig
  • Make sure you're using the correct prefix (default is "/")
  • Check that AdminCommands.luau is in the correct location
