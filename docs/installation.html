‚öôÔ∏è Configuration

Basic Configuration

Edit the ChatConfig.luau file to customize basic settings:

Chat Settings

-- Basic chat configuration
Config.MaxMessageLength = 200
Config.MessageCooldown = 1 -- seconds
Config.EnableEmojis = true
Config.EnableMarkdown = false

UI Customization

-- UI appearance settings
Config.Theme = {
    PrimaryColor = Color3.fromRGB(99, 102, 241),
    BackgroundColor = Color3.fromRGB(255, 255, 255),
    TextColor = Color3.fromRGB(31, 41, 55)
}

Advanced Configuration

Configure advanced features and performance settings:

Performance Settings

-- Performance optimization
Config.MaxChatHistory = 100
Config.MessageBatchSize = 10
Config.UpdateInterval = 0.1
Config.EnableCompression = true

Feature Toggles

-- Feature configuration
Config.Features = {
    AdminCommands = true,
    PlayerMentions = true,
    ChatChannels = false,
    VoiceChat = false,
    FileSharing = false
}

Security Configuration

Configure security and moderation settings:

Rate Limiting

-- Rate limiting configuration
Config.RateLimit = {
    MaxMessages = 5,
    TimeWindow = 10, -- seconds
    PenaltyDuration = 30 -- seconds
}

Content Filtering

-- Content moderation
Config.Moderation = {
    EnableFilter = true,
    StrictMode = false,
    BannedWords = {},
    AutoMute = true
}

üîß Troubleshooting

Possible causes:

  • ChatGui.rbxm not properly imported to StarterGui
  • Client script not running
  • Script errors preventing initialization

Solutions:

  • Check the Output window for errors
  • Verify all files are in correct locations
  • Restart the test server

Possible causes:

  • Server scripts not running
  • Signal system not initialized
  • Rate limiting blocking messages

Solutions:

  • Check ServerScriptService for errors
  • Verify GoodSignal module is accessible
  • Test with slower message sending

Possible causes:

  • Player doesn't have admin permissions
  • AdminCommands script not running
  • Incorrect command syntax

Solutions:

  • Add your UserId to the admin list
  • Check command syntax in documentation
  • Verify AdminCommands.luau is in ServerScriptService
